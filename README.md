# 使用方法：

1. 现在本地址内Clone代码压缩包到本地文件夹，再进入WordPress管理后台，进入菜单：插件->上传插件，根据提示上传插件包

2. 上传成功后，在插件页面启用该插件

3. 进入工具 -> 微信小程序设置，分别填入对应的小程序appID、secretkey以及其他相关设置，保存

# 使用说明：

插件安装启用后，需要设置文章分类目录：微信小程序封面，设置方法是进入wordpress管理后台，进入菜单：文章->分类目录，在微信小程序封面输入图片的链接地址即可
